# Death Counter Popup

A simple mod to add a text popup when you die, showing how many times you died at that percentage (like `97x2`).

This mod saves death counter per level, and will initialize from Death Tracker if it's installed.
One advantage of using alongside Death Tracker is the mod's level linking feature.
This mod will NOT link levels, and the popup will be per level.
